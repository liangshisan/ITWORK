# 梁十三 IT 运维智能体 · 同事上手说明

> 给同事看的极简版，完整结构见 `README.md`。

## 一句话

对着智能体说话，它帮你自动提飞书工单、做资产出入库、查/开邮箱、推向日葵远程码。

## 三步跑起来

**1. 下载仓库**

```bash
git clone https://github.com/liangshisan/ITWORK.git
```

**2. 装成 WorkBuddy 专家包**

把仓库内容放进 WorkBuddy 的专家包目录（一般在你用户目录下的
`.workbuddy/plugins/marketplaces/my-experts/plugins/liang-shisan-agent/`），
让 WorkBuddy 重新加载即可。详细命令看 `README.md` 的「方式一」。

**3. 启动带调试端口的 Edge（必须）**

工单填写、金山文档出入库都需要控制浏览器，必须让 Edge 开着远程调试端口：

- 仓库里已放了 `Edge-Debug-9222.lnk`
- 把它**复制到桌面，双击**启动（放别处双击也行）
- 它会开一个带 `--remote-debugging-port=9222` 的 Edge 窗口
- 智能体就能连上它自动填表了

> ⚠️ 没开这个端口，飞书工单 / 金山文档自动化会连不上浏览器，直接歇菜。

## 日常怎么用

打开 WorkBuddy → 选「梁十三智能体」→ 直接说：

- “提个工单，张三的电脑黑屏了”
- “出库一台笔记本，领用人李四”
- “查一下 zhangsan 的邮箱开没开”
- “发一下远程码”

## 环境要求

- Windows 10 / 11
- 已装 Microsoft Edge（默认安装位置即可，快捷方式指向 `C:\Program Files (x86)\Microsoft\Edge\...`）
- Python 3.10+，技能脚本依赖 `playwright` / `pyautogui` / `Pillow`
- 内网环境（飞书、金山文档、AD、ITDB、QSSO 才能访问）
